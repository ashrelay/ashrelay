I write code that works, mostly so I don't have to look at it again later. 

`>` **Current focus:** Architecting solutions that outlive my patience for them.<br>
`>` **Learning:** The subtle difference between a feature and a beautifully documented bug.<br>
`>` **Collaboration:** Open to ideas, provided they come with a reproducible test case.<br>
`>` **Inquiries:** System design, performance bottlenecks, and why DNS is usually the problem.<br>
